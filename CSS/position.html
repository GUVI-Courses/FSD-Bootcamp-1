Static : The Element is positioned according to the normal flow of the document





Relative: The element is positioned relative to its normal position. Setting the top, right, bottom, and left properties will cause it to be adjusted away from its normal position.





Absolute: The element is positioned relative to its closest positioned ancestor (not static). If there are no such ancestors, it uses the document body, and moves along with page scrolling.




Fixed: The element is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled.




Sticky: The element is treated as relative positioned until it crosses a specified threshold, at which point it is treated as fixed positioned.
